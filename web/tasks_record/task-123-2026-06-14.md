# Task 123：demand-detail.html 页面手机版悬停区域调整

## 任务内容

demand-detail.html 页面手机版，顶部需求卡片悬停时只展示设计图 image-169.png 中的部分，其余信息隐藏。

## 修改摘要

- 调整 `demand-detail.html` 手机版顶部需求卡片悬停表现，悬停后仅保留用户信息、品牌/服务、标签、状态角标和 Edit 按钮。
- 移动端改为 JS 控制 fixed 悬停，规避原生 sticky 在慢速滚动临界点回弹到顶部的问题。
- 增加悬停占位节点，使用悬停后卡片高度占位，避免刚进入悬停时内容区域出现大段空白。
- 悬停切换时保护当前滚动位置，确保 1px 慢滚进入和退出悬停都不会跳动。
- 更新 `demand-detail.css`、`demand-detail.js` 防缓存版本，并通过 `node scripts/build-pages.js` 重新生成根目录页面产物。

## 涉及文件
