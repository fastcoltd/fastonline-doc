# Task 106：brand.html页面的手机版需要修改

## 任务内容

brand.html 页面的手机版，所有 item 的收藏按钮收藏状态都要改成红色，并检查全局收藏按钮的收藏状态颜色都为红色。

## 修改页面

- `brand.html`

## Diff 总结

- 将 `src/partials/components/brand-hot-item-card.html` 中 brand Hot items 的收藏按钮状态改为已收藏：
  - `like` 从 `0` 改为 `1`
  - `like_icon` 从 `image/Vector_nor.svg` 改为 `image/Vector_sel.png`
  - `aria_pressed` 从 `false` 改为 `true`
- 在 `css/common.less` 增加移动端全局已收藏按钮规则，并同步到 `css/common.css`：
  - `.icon-aixin[aria-pressed="true"]`
  - `.icon-aixin[data-like="1"]`
  - 强制 `filter: none !important;`，避免已收藏红色图标被其他移动端样式滤成白色。
- 更新 `src/pages/brand.html` 的 CSS 防缓存版本参数。
- 执行 `node scripts/build-pages.js`，同步生成根目录页面产物，并刷新根目录 HTML 的缓存版本号。

## 检查结果

- `node scripts/build-pages.js` 执行成功。
- `node --check scripts/build-pages.js` 检查通过。
- 已确认生成后的 `brand.html` 中 20 个 brand Hot item 均为 `Vector_sel.png` 红色已收藏状态。
