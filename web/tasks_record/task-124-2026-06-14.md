# Task 124：brand-all.html 页面手机版 item 宽度与 Figma 样式调整

## 任务内容

brand-all.html 页面手机版，每个 brand item 原本只占屏幕宽度的一小部分，需要撑满屏幕宽度。后续按 Figma 设计稿节点 `859:37677` 调整手机版 item 样式。

## 修改摘要

- 移除 `brand-all.html` 手机版 `.brand-item` 的 `max-width: 32.1rem` 限制，并设置 `flex: 0 0 100%`，让每个 item 撑满列表可用宽度。
- 按 Figma 调整手机版 brand item 卡片样式：圆角、内边距、阴影、卡片间距。
- 按 Figma 调整手机版 logo 区、标题、描述、标签和底部统计区的尺寸、字号、行高、字体与间距。
- 仅在 `max-width: 768px` 媒体查询内修改样式，避免影响 PC 版。
- 同步修改 `css/brand.less` 与 `css/brand.css`，并通过 `node scripts/build-pages.js` 重新生成根目录页面产物，刷新资源 cache version。

## 涉及文件

- `css/brand.less`
- `css/brand.css`
- `brand-all.html`
- 根目录其他 HTML 构建产物：由 `node scripts/build-pages.js` 统一刷新资源 cache version。
- `codex_tasks/TASKS.md`
