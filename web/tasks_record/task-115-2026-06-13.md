## Diff 总结

- 在 `css/store.less` 中调整 `store-all` 手机版 Vertical 和 Horizontal item 的文字样式。
- 同步更新 `css/store.css`，保持 Less 源码与 CSS 产物一致。
- Vertical 模式 item 的标题、描述、评分、Brand、Service、标签文字对齐 Horizontal 模式字号规格：
  - 标题：`2rem / 2.6rem`，`font-weight: 600`

  - 评分数量：`1.5rem / 2.2rem`
  - 标签文字：`1.25rem / 1.5rem`
- Horizontal 模式 item 补齐 `Inter` 字体、字号、行高和评分文字规格。
- 运行 `node scripts/build-pages.js` 重新生成根目录 HTML，并刷新 CSS/JS 防缓存版本号。
