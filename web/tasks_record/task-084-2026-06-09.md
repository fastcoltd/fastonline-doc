# Task 084：brand.html 手机版 Demands item 字号与裁切调整

## 任务内容

`brand.html` 手机版 Demands tab 的 item 中部分文字太小，需要参考 Content tab 的 item 字号调整。

## 修改页面

- `brand.html`

## 修改内容

- 调整 `css/detail/brand.less` 中品牌详情页手机版 Demands item 的字段文字：
  - Brand、Service、Price、Total price 等字段从 `1.2rem / 1.8rem` 调整为 `1.5rem / 2.25rem`，与 Content tab item 的移动端字段字号一致。
  - 同步调整字段行高最小值，避免文字放大后垂直裁切。
- 放开 Demands item 内部展示限制：
  - 卡片从固定高度改为自适应高度，并保留最小高度。
  - `demand-item-content` 改为使用完整内容宽度。
  - tag 区域取消固定高度和 `max-height`，允许完整换行展示。
  - 字段值允许完整展示，不再被 `overflow` 裁切。
- 针对 `Total price` 值单独保持单行展示，避免 `$1926.00` 在仍有可用空间时被拆行。
- 同步更新 `css/detail/brand.css`。
- 通过 `node scripts/build-pages.js` 重新生成根目录静态页面，刷新资源版本防缓存。
