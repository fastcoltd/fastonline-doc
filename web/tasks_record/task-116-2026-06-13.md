# Task 116：item-detail.html 页面手机版顶部图片修复

## 任务内容

`item-detail.html` 手机版顶部图片不见了，需要在未滚动时展示顶部图片；滚动到悬停状态时，悬停部分不展示图片。

## 修改页面

- `item-detail.html`

## Diff 总结

- 在 `css/detail/item-detail.less` 和 `css/detail/item-detail.css` 中调整 `item-detail` 手机版顶部购买区域样式：
  - 未进入悬停状态时显示 `.item-detail-purchase-image`。
  - 进入悬停状态后隐藏顶部图片。
  - 覆盖移动端原生 `position: sticky`，配合 JS 使用 `fixed`，避免图片显隐造成慢速滚动回弹。
  - 覆盖全局 `.layout-vertical` 对该购买容器带来的左外边距。
  - 将顶部商品容器、图片、内容和购买区域左右边界对齐到顶部搜索框/面包屑。
- 在 `js/detail/item-detail.js` 中调整移动端悬停逻辑：
  - 基于初始顶部位置判断是否进入悬停。
  - 移动端悬停时给正文补充顶部占位，避免 fixed 后文档流高度变化。
  - resize 时重新计算悬停阈值。
- 执行 `node scripts/build-pages.js` 重新生成根目录 HTML，并刷新本地 CSS/JS 资源防缓存版本号。

## 验证

- `node scripts/build-pages.js` 通过。
- `node --check js/detail/item-detail.js` 通过。
- 使用 Chrome DevTools/CDP 移动端视口验证：
  - 慢速上滑无回弹。
  - 未悬停时顶部图片显示，悬停后顶部图片隐藏。
  - 顶部商品容器、图片、内容、购买区域与搜索框/面包屑左右边界一致。
