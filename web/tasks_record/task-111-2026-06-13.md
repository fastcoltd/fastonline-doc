# Task 111 - blog.html 手机版 Follow 按钮位置调整

## 任务内容

`blog.html` 页面的手机版中，Follow 按钮需要放在与 Hot 标签水平居中的最右侧。

## 修改页面

- `blog.html`

## 修改内容

- 在 `css/detail/blog.less` 中调整移动端博客头部布局：
  - 将 `.blog-header` 设为相对定位；
  - 将 `.right-part-wrapper` 绝对定位到标题首行右侧；
  - 使用 `top: 1.375rem` 和 `transform: translateY(-50%)` 让 Follow 按钮与 Hot 标签所在行垂直居中；
  - 给 `.blog-top-wrapper-title` 增加右侧预留宽度，避免按钮遮挡标题和标签；
  - 让 `.blog-info-wrapper` 占满剩余宽度，保证窄屏下内容正常换行。
- 同步更新 `css/detail/blog.css`，保持 `.less` 和 `.css` 一致。
- 执行 `node scripts/build-pages.js` 重新生成根目录静态页面，并刷新资源防缓存版本号。

## 验证

- `node scripts/build-pages.js` 执行通过。
- `node --check scripts/build-pages.js` 执行通过。
- `node --check js/blog.js` 执行通过。
- 按要求未调用 chrome mcp。
