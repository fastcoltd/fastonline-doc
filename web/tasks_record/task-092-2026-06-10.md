# Task 092 - demand-detail.html 页面 PC 版顶部 sticky 右侧状态标签对齐

## 任务内容

`demand-detail.html` 页面 PC 版需要修改：顶部内容在悬停后，最右上角 `Delivered` 状态标签区域需要和下面 bidding item 的右侧视觉表现一致。

## 修改内容

- 在 `src/pages/demand-detail.html` 中为顶部 summary 卡片的 `Delivered` 状态标签增加专用裁切容器 `demand-summary-status-clip`。
- 在 `css/detail/demand-detail.less` 和 `css/detail/demand-detail.css` 中新增状态标签裁切层样式，让 `Delivered` 标签在普通状态和 sticky 状态下保持同一位置与同一裁切效果。
- 调整 `.demand-summary-card.is-sticky`：
  - 保持 `border-radius: 0`，悬停/置顶时顶部不出现圆角。
  - 使用 `border: 1px solid transparent`，保留边框占位但视觉透明，避免 sticky 状态布局跳动。
- 运行 `node scripts/build-pages.js` 生成根目录 HTML，并刷新 css/js 缓存版本号。

## 验证

- `node scripts/build-pages.js`
- `node --check scripts/build-pages.js`
- `node --check js/detail/demand-detail.js`

## 主要影响文件

- `src/pages/demand-detail.html`
- `demand-detail.html`
- `css/detail/demand-detail.less`
- `css/detail/demand-detail.css`
- 根目录其他 HTML 文件仅由构建脚本刷新 css/js 缓存版本号。
