# Task 122：全局的评分组件，传值确认和修改

## 任务内容

需要确认全局的评分组件，即例如 `post-detail.html` 中的评分展示，传入的是分数还是百分比。如果传递的是百分比，改成传递分数，满分 5 分，传递分数后展示对应的分数和星星。

## 修改总结

- 将通用评分组件 `rating-score-only.html`、`rating-with-count.html` 的星星填充方式从 `style="width: xx%"` 改为 `style="--star-fill: xx%"`。
- 在 `scripts/build-pages.js` 中新增评分填充派生逻辑，根据 5 分制分数自动生成 `score_fill` / `rating_score_fill` / `star_score_fill`。
- 移除源码 include 中的 `width`、`rating_width`、`star_width` 等百分比评分参数，统一保留 `score` / `rating_score` / `star_score` 分数参数。
- 将 `blog.html`、`post-detail.html`、`item-detail.html`、`system-post-detail.html` 中手写的评分星星结构改为复用评分组件。
- 同步修正 JS 动态模板中的评分星星写法，避免运行时插入的内容继续使用 `width` 百分比。
- 执行 `node scripts/build-pages.js` 重新生成根目录页面，并刷新 css/js 防缓存版本。

## 验证

- `node scripts/build-pages.js` 通过。
- `node -c scripts/build-pages.js` 通过。
- 已扫描确认源码、JS、根目录产物中没有残留 `stars-inner style="width: ..."` 写法。
- 已扫描确认源码 include 中没有残留评分百分比参数。
