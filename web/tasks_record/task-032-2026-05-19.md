# Task 032 - 2026-05-19

## 任务
点击 `index.html` 右上角 Join 出现的一系列弹框（登录/注册/找回密码/2FA 等）中，密码显示与隐藏按钮点击无效；并且 `signin-2fa.html`、`signin-join.html`、`signin-login.html`、`signin-reset-2fa.html`、`signin-reset-password.html` 页面中同类密码显示与隐藏按钮也需修复。

## 修改页面
- `src/partials/signin-modal.html`
- `js/login.js`
- `css/signin.less`
- `css/signin.css`
- 根目录构建产物页面（执行 `node scripts/build-pages.js` 后同步）

## 变更摘要
- 在 `src/partials/signin-modal.html` 中为登录/注册密码输入增加可点击显隐按钮结构：
  - 新增 `.signin-password-field`
  - 新增 `.signin-password-toggle`
- 在 `js/login.js` 中新增通用密码显隐逻辑 `initPasswordToggle(rootElement)`：
  - 统一绑定 `.signin-password-toggle` 点击事件
  - 点击后在 `password`/`text` 之间切换
  - 同步更新 `aria-label` 与可视状态类
- 在 `css/signin.less` 与 `css/signin.css` 中补充显隐按钮样式，且保持移动端优先生效：
  - 桌面端维持原视觉表现
  - 移动端（`max-width: 768px`）展示可点击按钮并提供显隐状态样式
- 按项目规则执行构建 `node scripts/build-pages.js`，更新根目录页面及资源防缓存版本参数（`?v=...`）。

## 影响范围
- 仅影响登录注册相关弹框与对应独立登录页面中的密码显隐交互（手机版）。
- 不影响其他业务功能与 PC 端布局。
