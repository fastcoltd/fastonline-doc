# Task 080 - index.html 手机版 Popular demands 文字大小

## 任务内容

- 页面：`index.html`
- 范围：手机版
- 要求：`Popular demands` 的 item 中指标文字太小，需要参考 `Hot campaigns` item 中同类指标文字大小调整，并保证布局合理美观。

## 修改摘要

- 仅在 `#popuar-demands-mobile` 移动端作用域内调整样式，避免影响 PC 版。
- 将 `Popular demands` item 指标文字从 `1.2rem / 1.8rem` 调整为 `1.4rem / 2rem`，接近 `Hot campaigns` 同类文字大小。
- 将指标内容区宽度从固定 `13.4rem` 改为 `100%`，为放大后的文字提供更多横向空间。
- 将内容区顶部间距从 `1rem` 调整为 `0.75rem`，避免卡片内部过度拥挤。
- 根据放大后的内容同步调整 item 卡片尺寸：
  - 宽度、最小宽度、横向滚动占位从 `16.5rem` 调整为 `18rem`。
  - 高度、最小高度从 `28.4rem` 调整为 `31.2rem`。
- 同步修改 `css/index.less` 和编译产物 `css/index.css`。
- 执行 `node scripts/build-pages.js` 重新生成根目录页面，并刷新本地 CSS/JS 防缓存版本号。
- 将 `codex_tasks/TASKS.md` 中对应任务状态标记为已完成。
