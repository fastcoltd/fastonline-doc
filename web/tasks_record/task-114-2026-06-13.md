# Task 114 - item-all.html 手机版 Horizontal 按钮宽度调整

## 任务内容

`item-all.html` 页面手机版 Horizontal 模式下的 item，`BUY NOW` 按钮需要撑满右侧剩余宽度，按钮右侧与收藏按钮右侧在垂直方向对齐。

## 修改摘要

- 修改 `css/item-all.less` 与同步产物 `css/item-all.css`：
  - `.item-all-horizontal-mobile-main` 从固定宽度改为 `flex: 1; min-width: 0;`
  - `.item-all-horizontal-mobile-meta` 从固定宽度改为 `width: 100%;`
  - `.item-all-horizontal-mobile-buy-btn` 从固定宽度改为 `width: 100%;`
- 执行 `node scripts/build-pages.js`，刷新根目录页面产物和资源防缓存版本号。
- 执行 `git diff --check`，未发现空白或语法类 diff 问题。

## 影响范围

- 仅调整 `item-all.html` 手机版 Horizontal item 的布局表现。
- PC 版样式未修改。
