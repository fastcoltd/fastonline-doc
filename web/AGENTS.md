# 无论什么情况下都必须遵守的约定
- always respond in 中文
- 本项目为静态页面项目，所有的网页都是静态的
  - 使用语义化 HTML
  - 使用原生 CSS，不要引入 UI 库
  - 按 Figma 的布局、字号、颜色、间距、圆角实现
  - 优先使用 flex / grid，不要大量 absolute 定位
- 项目需要适配pc端和手机端，以响应式布局，来调整pc端和手机端展示的样式
- 所有的页面中的内容，宽度都必须限制为 1440，展示在页面横向中央，举个例子，如果屏幕宽度是 1920，那么页面内容只有中间1440的宽度展示内容
- 所有的需求，都需要保证在不修改其他功能的前提下，以最小化修改来实现需求；只修改需求中指定的内容，不要修改其他任何内容
